# App Screenshots

Required before store submission.

## Apple App Store

| Device | Size | Count |
|--------|------|-------|
| iPhone 6.9" (15 Pro Max) | 1320×2868 | min 3 |
| iPhone 6.7" (14 Plus) | 1284×2778 | min 3 |
| iPad Pro 13" | 2064×2752 | min 3 (if iPad supported) |

## Google Play Store

| Type | Size | Count |
|------|------|-------|
| Phone | 1080×1920 min | min 2, max 8 |
| Tablet 7" | 1080×1920 min | optional |
| Feature Graphic | 1024×500 | 1 required |

| Mobile screenshot | 390×844 |
